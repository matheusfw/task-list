# Sistema de Tarefas em PHP

Projeto simples feito em PHP puro para praticar:

- Programação Orientada a Objetos (POO)
- Organização de código
- Estrutura estilo MVC

## Funcionalidades

- Criar tarefas
- Listar tarefas
- Marcar como concluída
- Contar tarefas concluídas

## Como rodar

1. Baixe o projeto
2. Execute o arquivo `index.php` em um servidor local (XAMPP, etc)
